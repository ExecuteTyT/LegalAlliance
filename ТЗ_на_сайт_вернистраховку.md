# ТЕХНИЧЕСКОЕ ЗАДАНИЕ
## Разработка сайта вернистраховку.рф

---

## 1. ОБЩАЯ ИНФОРМАЦИЯ О ПРОЕКТЕ

**Название сайта:** вернистраховку.рф  
**Тип сайта:** Одностраничный лендинг (Landing Page) + страница "Услуги"  
**Целевая аудитория:** Физические лица, взявшие кредиты с навязанными страховками и дополнительными услугами  

**Основная цель сайта:**  
Максимальная конверсия посетителей в заявки (лиды) через:
- Сбор номеров телефонов
- Заполнение форм обратной связи
- Звонки по телефону

**Ключевой принцип:** ПРОСТОТА + ДОВЕРИЕ + СРОЧНОСТЬ

---

### 📍 БЛОК 1: ШАПКА (HEADER) - STICKY

**Назначение:** Постоянный доступ к контактам и CTA

**Структура:**
```
┌──────────────────────────────────────────────────────────────┐
│ [ЛОГОТИП]        [Меню]              [☎ 8-900-123-45-67]     │
│ вернистраховку.рф  • Услуги          [КНОПКА: Бесплатная     │
│                    • Как работает     консультация]           │
│                    • Отзывы                                   │
│                    • Контакты                                 │
└──────────────────────────────────────────────────────────────┘
```

**Элементы:**

1. **Логотип** (слева)
   - Текст: "вернистраховку.рф"
   - Иконка: щит с галочкой или стрелка возврата
   - Кликабелен (ведет наверх)

2. **Меню навигации** (центр)
   - Якорные ссылки на блоки
   - Плавная прокрутка (smooth scroll)
   - На мобильных: бургер-меню

3. **Телефон** (справа)
   - Крупный шрифт, жирный
   - Иконка телефона
   - Кликабельный (tel: ссылка)
   - Рабочее время под номером мелким шрифтом

4. **CTA-кнопка** (справа)
   - Текст: "Бесплатная консультация" или "Узнать сумму возврата"
   - Яркий цвет (оранжевый/зеленый)
   - При клике: открывается модальное окно с формой

**Поведение:**
- Фиксированная позиция (sticky) при скролле
- Тень появляется при прокрутке вниз
- На мобильных: уменьшенная версия (только лого + телефон + бургер)

│                          с деньгами или юристы]              │

**Элементы:**

1. **Заголовок H1** (самый крупный текст на сайте)
   - Текст: "Вернём навязанные страховки и услуги по кредиту"
   - Альт: "Верните деньги за ненужную страховку по кредиту"
   - Шрифт: жирный, 48-60px на десктопе
   - Цвет: темно-синий или черный

2. **Подзаголовок H2**
   - "БЕЗ ПРЕДОПЛАТЫ. Платите только после получения денег"
   - Можно выделить "БЕЗ ПРЕДОПЛАТЫ" другим цветом
   - Шрифт: 24-28px

3. **3 ключевых преимущества** (иконки + текст)

   - ✓ Работаем по всей России
   - ✓ Вернули уже 47 млн рублей клиентам

     * Телефон (обязательное, маска +7 (___) ___-__-__)
     * Сумма кредита (необязательное, для расчета)
   - Кнопка: "РАССЧИТАТЬ ВОЗВРАТ" или "ПОЛУЧИТЬ КОНСУЛЬТАЦИЮ"
   - Под формой: галочка согласия с политикой конфиденциальности

5. **Две CTA-кнопки** (если нет формы справа)
   - Первая: "Рассчитать возврат" (основная, яркая)
   - Вторая: "Бесплатная консультация" (вторичная)

6. **Изображение/иллюстрация**
   - Вариант А: Фото реальных юристов компании
   - Вариант Б: Довольные клиенты с деньгами
   - Вариант В: Иллюстрация "деньги возвращаются"

**Фоновое оформление:**
- Светлый градиент (белый → светло-голубой)
- Или изображение банка/документов с полупрозрачным оверлеем

### 📍 БЛОК 6: КАЛЬКУЛЯТОР ВОЗВРАТА (ИНТЕРАКТИВ)

**КРИТИЧЕСКИ ВАЖНО:** Это самый сильный инструмент лидогенерации!

**Назначение:** 
- Вовлечь пользователя
- Показать конкретную сумму возврата
- Получить контакты в обмен на расчет

**Структура:**
```
┌─────────────────────────────────────────────────────────────┐
│              УЗНАЙТЕ СУММУ ВОЗВРАТА ЗА 30 СЕКУНД             │
│                                                               │

│  │                                                         │ │
│  │ Сумма кредита:  [__________] руб.                      │ │
│  │                                                         │ │
│  │ Сумма страховки: [__________] руб.                     │ │
│  │                                                         │ │
│  │ Дата оформления: [__.__.____]                          │ │
│  │                                                         │ │
│  │ Досрочно погасили? [ ] Да  [ ] Нет                     │ │

│  │                                                         │ │
│  │  Укажите контакты, чтобы получить точный расчет:       │ │
│  │                                                         │ │
│  │  Ваше имя:    [________________]                        │ │
│  │  Телефон:     [+7 (___) ___-__-__]                     │ │
│  │  Email:       [________________] (необязательно)        │ │
│  │                                                         │ │
│  │  [✓] Согласен с политикой конфиденциальности           │ │
│  │                                                         │ │
│  │            [ПОЛУЧИТЬ ТОЧНЫЙ РАСЧЕТ]                     │ │
│  └────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────┘
```

**Логика работы:**

1. **Этап 1: Сбор данных о кредите**
   - Пользователь вводит данные о кредите
   - Кнопка "Рассчитать возврат"

2. **Этап 2: Показ примерной суммы**
   - Появляется примерный расчет (например, 40-50% от суммы страховки)
   - 🎉 "Вы можете вернуть ~ 45 000 ₽!"
   - Эмоциональный момент - человек видит ДЕНЬГИ

3. **Этап 3: Запрос контактов**
   - "Укажите контакты для точного расчета"
   - Поля: Имя, Телефон (обязательно), Email (опционально)
   - Кнопка: "Получить точный расчет"

4. **Этап 4: Отправка заявки**
   - После отправки: "Спасибо! Мы позвоним вам в течение 15 минут"
   - Или редирект на страницу благодарности

**Формула расчета (примерная):**
- Потребительский кредит: 30-50% от суммы страховки (если в течение 3 лет)
- Автокредит: 40-60% от КАСКО + 50-70% от страхования жизни
- Если досрочно погашен: до 80% от остатка страховки

**Дизайн:**
- Фон: яркий градиент (синий→фиолетовый или оранжевый→красный)
- Форма: белая карточка с тенью
- Результат: крупные цифры, зеленый цвет
- Анимация при появлении результата
