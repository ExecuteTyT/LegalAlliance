# Правовой Альянс | Банкротство физлиц

Веб-приложение для юридической компании, специализирующейся на банкротстве физических лиц.

## Технологии

- React 19
- TypeScript
- Vite
- Tailwind CSS
- Lucide React (иконки)

## Установка и запуск

```bash
npm install
npm run dev
```

## Сборка для production

```bash
npm run build
```

## Деплой на Vercel

Проект настроен для автоматического деплоя на Vercel. Просто подключите репозиторий к Vercel через веб-интерфейс или используйте Vercel CLI.
